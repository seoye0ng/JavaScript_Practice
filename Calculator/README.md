# 계산기 (CALCULATOR)

1. 숫자를 누르면 입력을 받는다.

   - 입력받은 값을 보여준다.(main_input, sub_input)

2. 연산자를 입력받는다.

   - 입력받은 값을 보여준다.(sub_input)
   - 연산자 버튼을 눌렀을 때 앞에 입력받은 값을 저장한다.(새로운 변수에)
   - 입력받은 연산자 값을 저장한다.(새로운 변수에)

3. 입력받은 값을 계산한다.

   - = 버튼을 눌렀을 때 저장한 값을 계산한다.
   - 저장한 값을 보여준다.

4. 리셋버튼을 누르면 값이 사라진다.

   - 저장된 값, 보여주는 값 모두 사라진다.

5. 백스페이스를 누르면 뒤에서부터 값이 지워진다.
   - 보여지는 값이 뒤에서부터 지워진다.
